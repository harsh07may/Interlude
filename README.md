# Interlude

Journal digitizer for turning handwritten interstitial journal pages into structured text.

## Development

```bash
pnpm install
pnpm run dev
```

## OCR

The app currently supports:

- Gemini API OCR using a Google AI Studio API key.
- Optional custom backend OCR endpoint for local or hosted experiments.

## Checks

```bash
pnpm run lint
pnpm run build
pnpm run test
```
